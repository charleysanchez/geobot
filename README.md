# geobot
Deep Learning model trained on Google Streetview images from around the world to pinpoint global coordinates based on image.

## Instructions

1. ```curl -L -o ~/Downloads/google-street-view.zip\
  https://www.kaggle.com/api/v1/datasets/download/paulchambaz/google-street-view && mv ~/Downloads/google-street-view.zip . && unzip google-street-view.zip
  ```

2. Run `scripts/organize_dataset.py`

## Dataset
[Kaggle Page](https://www.kaggle.com/datasets/paulchambaz/google-street-view/data)
